Pricipais formatações

Negrito / destaque

Forma de escrita (sem significado semantico)

Nessa frase temos um termo em negrito usando a tag <b></b>, que nao é semantica, só escreve o texto mais forte


Semantico

para escrever com as tags semanticas, negrito se usa a tag <strong> termo em destaque </strong> isso diz ao navegador e aos usuarios que aquele texto é especifico, e tambem deixa com o visual negrito

Itálico /Ênfase

Não semantico, sem sentido semantico, só diz a forma itálico

Nesta frase temos um termo em itálico, usando a tag <i>


Semantico

Nestra outra frase temos o termo semantico para dizer ao navegador que auqele trecho do texto é um italico ou uma enfase : usando a tah <em>Termo em Itálico</em>
EX: Termo em itàlilco

Texto Marcado ou Marca texto SEMANTICO

Podemos criar também um texto marcado utilizando a tag <mark> </mark>. E para alterar a cor do texto marcado utilizando Css na própria tag mark adicionando style=""
EX:<mark style="background-color: rgb(211, 154, 154);">texto marcado</mark>

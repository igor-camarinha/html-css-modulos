Reproduzindo audios em HTML5

Para adicionar um audio basta adicionar a tag <audio src="Caminho do arquivo" controls autoplay>

Caso não tenha conseguido ouvir o podcast! clique aqui para baixar .MP3 

                
                    <audio preload="metadata" controls autoplay >
                        <source src="midia/guanacast-33.mp3" type="audio/mpeg">
                        <source src="midia/guanacast-33.ogg" type="audio/ogg">
                        <source src="midia/guanacast-33.wav" type="audio/waw">
                        p>Caso não tenha conseguido ouvir o podcast! clique aqui para baixar .MP3 

                    </audio>

No código acima caso o navegador vai tentar carregar o primeiro <SOURCE> , caso nao haja compatibilidade, ira tentar o proximo.
